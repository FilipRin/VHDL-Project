Verilog project for Altera Cyclone (III and V) FPGA boards.
-----------------------------------------------------------------
Simulation part of project was testing how will ALU unit work with defined registers.
Synthesis part was programming a lot of different modules that together make complex microcontroller that executes instructions from mem_init.mif correctly. This was tested on real board, using switches as inputs while program executes instructions. All modules were made to follow documentation for project (Input/Output signals and even parameters).
